# Arc
An operating system written in *mostly* [SPWN](https://github.com/Spu7Nix/SPWN-language) *Hopefully.*


## Requirements
I suck at Makefile, so make sure you have the following.

- GRUB
- GCC
- NASM
- XORRISO
- LD
